## Callbacks (колбэки) для асинхронности

Колбэки - функции, которые передаются в другие функции как параметры и вызываются
после завершения асинхронной операции.

```ts
function fetchData(callback: (data: string)=>void) {
    setTimeout(() => {
        callback("Data is got");
    }, 1000);
}

fetchData((data) => {
    console.log(data);
});
```

Здесь у нас функция fetchData принимает в себя колбэк. Она имитирует выполнение сложной
работы (таймер на 1 секунду). Далее вызывается колбэк, в который передаются данные (это
может быть результат асинхронной операции, в нашем случае это строка). 

И далее мы вызываем fetchData и передаем в нее колбэк, в котором говорим, что делать, 
когда данные получены.
